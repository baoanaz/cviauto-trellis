# Cviauto Instructions

These instructions are for AI assistants working in this project.

This project is managed by Cviauto. The working knowledge you need lives under `.cviauto/`:

- `.cviauto/workflow.md` — development phases, when to create tasks, skill routing
- `.cviauto/spec/` — package- and layer-scoped coding guidelines (read before writing code in a given layer)
- `.cviauto/workspace/` — per-developer journals and session traces
- `.cviauto/tasks/` — active and archived tasks (PRDs, research, jsonl context)

If a Cviauto command is available on your platform (e.g. `/cviauto:finish-work`, `/cviauto:continue`), prefer it over manual steps. Not every platform exposes every command.

If you're using Codex or another agent-capable tool, additional project-scoped helpers may live in:
- `.agents/skills/` — reusable Cviauto skills
- `.codex/agents/` — optional custom subagents

Managed by Cviauto. Edits outside this block are preserved; edits inside may be overwritten by a future `cviauto update`.
